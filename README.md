# odoo-docker-compose
Odoo docker compose for easy usage

# HOW TO USE

- If you use Linux:
<code>
    docker compose --env-file .env.example -f odoo-docker-compose-linux.yml up -d
</code>

- If you use Windows
<code>
    docker compose --env-file .env.example -f odoo-docker-compose-win.yml up -d
</code>
